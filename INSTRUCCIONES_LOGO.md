# Instrucciones para agregar el logo

## Paso 1: Guardar el logo
1. Guarda la imagen del logo de ADICTION BOUTIQUE que te proporcioné
2. Nómbrala como `logo.png`
3. Colócala en la carpeta `public/images/logo.png`

## Paso 2: Verificar
El logo debe estar en: `public/images/logo.png`

## Formato recomendado:
- Formato: PNG con fondo transparente
- Tamaño recomendado: 400x200 pixels (o proporcional)
- Peso: Menos de 100KB para carga rápida

## El logo se usa en:
- Tickets de venta (POS)
- PDFs de tickets
- Emails de tickets
- Interfaz del sistema
