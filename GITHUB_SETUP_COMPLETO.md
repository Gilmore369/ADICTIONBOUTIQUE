## 🚀 Próximos Pasos

### Para Colaboradores

1. **Clonar el repositorio**:
```bash
git clone https://github.com/Gilmore369/ADICTIONBOUTIQUE.git
cd ADICTIONBOUTIQUE
```

2. **Instalar dependencias**:
```bash
npm install
```

3. **Configurar variables de entorno**:
```bash
cp .env.local.example .env.local
# Editar .env.local con tus credenciales
```

4. **Ejecutar migraciones de Supabase**:
   - Ir a tu proyecto de Supabase
   - Ejecutar los archivos en `/supabase/migrations/` en orden

5. **Iniciar desarrollo**:
```bash
npm run dev
```

### Para Producción

1. **Configurar Vercel/Netlify**:
